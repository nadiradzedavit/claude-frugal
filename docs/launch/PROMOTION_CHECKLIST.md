# Launch Checklist

## Before Posting

- `npm test` passes.
- README install command works.
- GitHub Discussions are enabled.
- Create a first Discussion: "Beta tester reports".
- Create a release tag after CI is green.
- Record a short terminal GIF or video.

## Posting Order

1. Post to a Claude Code or Claude community asking for beta testers.
2. Post Show HN only after installation has been tested from a fresh clone.
3. Share a short social thread with beta framing.

## Response Rules

- Thank people for negative results.
- Ask for exact workflow details.
- Do not defend theoretical numbers as proven.
- Open issues immediately for reproducible failures.

## Metrics To Track

- Stars.
- Clone/install failures.
- Beta reports opened.
- GitHub traffic sources.
- Common complaints.
